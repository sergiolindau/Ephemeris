The file datetime.js implements a javascript Calendar object from Mozilla® Developer Network web docs and others providing properties and methods defined in ECMAScript® 2018 Language Specification standard for Calendar calculations.

This file (datetime.html) is a documentation and test portion for datetime.js library.

Dependencies

This file depends of mathext.js. Documentation and test suite at mathext.html

Usage

To use this extensions include in the header section of your HTML file the script include:

<script language="javascript" src="mathext.js"></script>
<script language="javascript" src="datetime.js"></script>

Description

Calendar is a defined object that has properties and methods for calendar constants and functions. Not a function object, is a const object.

The Calendar const can't be extend.

Notes: See Implementation for navigator dependant details.
